Introduces support for "Multiple top-level domains" and "Display Advertiser Support" into the Standard dashboard.

== Description ==

"Standard Advanced Google Analytics" introduces two new fields in to Standard 'General' tab section that makes it easy to add support for multiple top-level domains, and to display advertiser support.

== Installation ==

= In The WordPress Dashboard =

1. Navigate to the 'Add New' plugin dashboard
2. Select `standard-advanced-google-analytics.zip` from your computer
3. Upload
4. Activate the plugin in the WordPress Plugin Dashboard

= Using FTP =

1. Extract `standard-advanced-google-analytics.zip` to your computer
2. Upload the `standard-advanced-google-analytics` directory to your wp-content/plugins directory
3. Navigate to the WordPress Plugin Dashboard
4. Activate the plugin from this page
